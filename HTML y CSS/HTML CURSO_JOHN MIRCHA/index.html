Etiquetas semanticas Estructurales

div-Etiqueta contenedora que no tiene semantica
header-Cabecera de un sitio web o seccion web
main-Define la seccion principal de un documento solo debe existir una etiqueta main por documento (como tambien un h1)
footer-Pie de pagina de un sitio web o seccion web
aside-Representa contenido complementario o segundario
section- Representauna s eccionde contenido generico
address- Representa una direccion fisica

Etiquetas de bloque vs Etiquetas de linea

La etiqueta de bloque por excelencia es div. Una etiqueta de bloque es aquella en la que sus saltos de linea se hacen con su etiquetas hermanas (Ocupan todo el lugara de la seccion donde estan escritas)
Otra etiqueta div. notar que hay una tabulacion
La etiqueta de linea por excelencia es span. los saltos de lienas NO se hacen mediante sus etiquetas hermanas (solo ocupa el espacio necesario que necesita) Otra etiqueta de linea. notar que NO hay tabulacion ⏫
